# Layout Standards

这是 `xsls-exam-skill` 的排版硬标准。

## 1. 纸张规格

- 默认纸张：`A3`
- 兼容规格：`大8开`
- 不要默认按 `A4` 竖版试卷处理

建议理解：

- 在高校模板和教务处文件里，常写 `A3`
- 在老师、印刷和民间习惯里，常说 `大8开`
- 实务中可视为同一档试卷规格的两个常见表述

## 2. 页面方向

- 默认 `landscape` 横版
- 一页优先视为两个半页：`left half + right half`

## 3. 装订线

- 默认在左侧
- 装订线不能紧贴纸边
- 需要给纸边与装订线之间留出明显空隙

## 4. 版心秩序

第一页通常优先包含：

- 标题区
- 分值表
- 选择题
- 填空题
- 第一部分解答题

后续页通常优先包含：

- 剩余解答题
- 带图题
- 长题

## 5. 有图题

- 图与题干、小问优先放在同一半页
- 立体几何、解析几何、三角函数配图题，不要把图甩到别页
- 若图占比过大，优先给该题独立半页

## 6. 不该做的事

- 不要为了塞内容，先把字号和行距压得过度紧凑
- 不要把整套试卷错误拆成多张 A4 纵向页面
- 不要为了一模一样而放弃成卷规范
