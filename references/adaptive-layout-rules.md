# Adaptive Layout Rules

这个文件处理“题量和题型变化时，如何仍然排得像正规考试卷”。

## 总原则

优先级从高到低：

1. 保证纸张规格与成卷方向正确
2. 保证装订线与版心秩序正确
3. 保证题目完整，不随意切碎
4. 保证题型块分配自然
5. 最后才是微调局部密度

## 1. 先按块排，不按单题排

把内容先分成块：

- 标题区
- 分值表
- 选择题块
- 填空题块
- 解答题块
- 图题块

优先移动整块，而不是一上来压每一道题的行距。

## 2. 题量变化时的规则

### 选择题变多

- 先尝试把整块选择题继续留在当前半页
- 若明显溢出，优先把后续题型块后移，而不是把选择题挤得极密
- 若仍放不下，允许选择题跨到下一个半页，但题型标题要保留清晰边界

### 填空题变多

- 优先与短解答题共享半页
- 若填空题变长，优先给填空题更多半页空间，再后移解答题

### 解答题变多或变长

- 解答题可以独占半页或整页
- 一道长题尽量不要拆到两个半页

## 3. 图题规则

- 立体几何、解析几何、函数图像题优先“文字在上，图在下”或“文字在左，图在右”
- 图题优先独立半页
- 图形若来自原样卷裁切，必须在 `review.md` 记录来源

## 4. 密度微调顺序

只有在块分配完成后，才允许做以下微调：

1. 题间距
2. 选项上下距离
3. 标题区上下间距
4. 分值表尺寸
5. 局部页边距

不建议轻易改动：

- 主字号
- 装订线位置逻辑
- 横版双半页结构

## 5. 判断排版是否失败

出现以下情况时，视为失败，需要重新分块：

- 某题底部被裁切或几乎看不见
- 半页底部留白特别大，但上方内容又过密
- 图题和图分离
- 看起来像普通文档，而不像正式试卷
