# Layout Detection Rules

这个文件定义“上传样卷图片或 PDF 后，先自动识别什么”。

## 目标

先识别整卷结构骨架，再决定怎么分页。

不要先做：

- 直接 OCR 全文后塞进模板
- 没判断纸张和方向就开始排
- 没判断图题位置就先生成 PDF

## 识别顺序

### 1. 识别纸张档位

优先判断：

- 更接近 `A3`
- 更接近 `大8开`
- 或者只是扫描出来的近似比例

只要整体逻辑属于“正规考试卷横版”，就优先落到 `A3/大8开` 规范里。

### 2. 识别页面方向

优先判断：

- 是横版还是竖版
- 是否属于“一页两半”的成卷逻辑

若样卷看起来像：

- 左半选择题
- 右半填空题/解答题

则应优先判定为“横版双半页”，而不是多张 A4。

### 3. 识别题型块

至少要先识别这些块：

- 标题区
- 分值表
- 选择题块
- 填空题块
- 解答题块
- 图题块

### 4. 识别特殊题

重点标记：

- 长选项题
- 立体几何题
- 解析几何图题
- 三角函数配图题
- 需要大图或表格的题

这些题在分页时优先级更高，不能按普通短题处理。

## 产物

识别完成后，必须先产出 `pagination-plan.md`，至少包含：

- 使用 `A3` 还是 `大8开`
- 是否横版双半页
- 每个半页准备放哪些题型块
- 哪些题独占半页
- 哪些题图文必须同半页
