# Auto Pagination Rules

这个文件定义“识别完样卷骨架后，怎么自动做分页决策”。

## 核心思想

自动分页不是“自动把空白填满”，而是：

- 先判断结构
- 再决定块分配
- 最后才做局部微调

## 决策顺序

### 1. 先定纸张与方向

- 优先 `A3`
- 如有需要切换 `大8开`
- 默认横版
- 默认一页两个半页

### 2. 再定半页块分配

常见优先级：

1. 标题区 + 分值表
2. 选择题块
3. 填空题块
4. 解答题块
5. 图题块

### 3. 再定特殊题落位

优先独立半页的情况：

- 图题明显占空间
- 解答题过长
- 题目带大表格
- 图和文字如果分开会影响阅读

### 4. 最后做局部压缩

只允许少量调整：

- 题间距
- 选项上下距离
- 标题和分值表空隙
- 图文相对位置

不应先动：

- 主字号
- 纸张规格
- 横版双半页结构
- 装订线逻辑

## 判断自动分页是否成功

满足以下几点才算成功：

- 没有题目被挤到看不见
- 没有明显不合理的大块空白
- 图题和图没有分离
- 视觉上仍像正规考试卷，而不是普通讲义
